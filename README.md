# 📌 Mini Lista de Produtos

Projeto desenvolvido durante meus estudos de **JavaScript**, com foco em manipulação do **DOM**, **arrays**, **objetos** e **eventos**.

---

## 📖 Sobre o Projeto

Este projeto permite ao utilizador adicionar produtos a uma lista informando **nome** e **idade**.

Os dados são armazenados num **array de objetos** e exibidos dinamicamente na interface.
Também é possível **remover itens da lista** e impedir a inserção de **produtos duplicados**.

---

## 🚀 Funcionalidades

* Adicionar produtos à lista
* Validação de campos vazios
* Bloqueio de produtos duplicados
* Remover itens da lista
* Atualização dinâmica da interface
* Limpeza automática dos inputs após adicionar
